🐛 Troubleshooting

Common Issues

  • 500 Error: Check PHP error logs, ensure .env file is correctly configured
  • 404 on API: Verify .htaccess files are uploaded and mod_rewrite is enabled
  • CORS Errors: Check that CORS headers are being set correctly
  • Database Errors: Verify database credentials and run initialization script
