# Is It Done Yet Frontend

React + Vite frontend for `apps/isitdoneyet`, aligned with the shared WebHatchery frontend workflow.

## Scripts

- `npm run dev`
- `npm run lint`
- `npm run type-check`
- `npm run test:run`
- `npm run build`
- `npm run ci`

## Environment

Create `.env` from `.env.example` if you need a non-default API base.

- `VITE_API_BASE_URL` (default: `/api`)
